# Summary

Yet another enclosure for the Ender3 Octoprint implementation with the Raspberry Pi 3 B/B+. This puppy is a clean and easy fit with the Pi and 30mm cooling fan. The design avoid to force the Pi connectors (and risking breaking them off) in order to install the pcb. The Cover/Housing design fit the connector neatly as it separate in half at the connector center.

Mounting on Ender 3:
Slide on the Square extrusion using 2 rail. Very snug fit (file down slots between duftail and enclosure for a looser fit).

Hartware:
Enclosure: M3X6mm (use M3-0.5 Tap) or #4 x3/8 self tap screw
Fan: #6 X 3/8 (use 6-32 Tap in Fan frame) or #6 x3/8 self tap screw
