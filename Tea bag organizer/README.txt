Summary:
[2025.12]Not a pre-cut design anymore. I'm leaving the "remixed" notion and all the old versions, because it's legacy and will be probably difficult to remove anyway, but I've completely redone it in OpenSCAD and created a customizable version. Keep the default values to get something very close to the earlier pre-cut versions, but it is not 100% compatible. The customizable version is up, remember to read the parameters description and don't create something that won't work! Margin for cover wall is the extra space inserted between the inside wall of the cover and the external wall of the matching cutout on the tea box, if you want the cover to sit very relaxed - increase it, since I've set the default as I use it, for putting the box vertically. Mind that it's not accounted for in the box itself, so it makes the cover slightly larger than the box. Since I didn't want wide wasteful walls, it is a good tradeoff.[Legacy description]Generic box parameterized by Customizer, then manually sized, modified and cut for better fit / beautified - also saving material. Holds a ton of tea bags. Cover holds on nicely and should not be hard to remove.[edit] Turns out, when I made it, my Blender-foo wasn't that strong. So following the requests to create more configurations, I've also fixed the original one to simplify it and avoid structural errors (even though slicers are smart enough to avoid these errors themselves).Please note the 3 different configurations for both the box and the cover files, the original one on the photos is 3x2.Also, please remember that the sizes are calculated for 0.4mm printing head, so use accordingly.I'll appreciate if anyone can share makes from other configurations for me to add to the description.
